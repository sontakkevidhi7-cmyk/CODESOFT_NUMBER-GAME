# Number Guessing Game

## Description
This is a simple Java-based Number Guessing Game. The program generates a random number between 1 and 100, and the user has to guess the correct number within a limited number of attempts.

The game provides hints if the guess is too high or too low. It also includes a scoring system and allows the user to play multiple rounds.

## Features
- Random number generation between 1 and 100
- User-friendly guessing interface
- Feedback for high and low guesses
- Limited number of attempts
- Score calculation based on remaining attempts
- Multiple rounds support
- Final score display

## Technologies Used
- Java
- Scanner Class
- Random Class

## How to Run

1. Save the file as `NumberGame.java`
2. Open Command Prompt or Terminal.
3. Navigate to the project folder.
4. Compile the program:
